## Live vs Persistant

| Mode            | Traits                                |
| --------------- | ------                                |
| Live Mode       | - No persistant storage               |
| ^^              | - Performance hit due to being in ram |
| ^^              | + Installation media                  |
| ^^              | + Restoration media                   |
| ^^              | + Emergency file system access        |
| Persistant Mode | + High efficiency                     |
| ^^              | + Persistant data                     |
| ^^              | + Installation is complete            |
